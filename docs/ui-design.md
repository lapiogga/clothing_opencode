# 화면설계서

> 군 피복 구매관리 시스템 UI/UX 설계 문서

## 디자인 시스템

### 색상 팔레트

| 이름 | 코드 | 용도 |
|------|------|------|
| Primary | `#3b82f6` | 주요 버튼, 링크 |
| Primary Dark | `#2563eb` | 호버 상태 |
| Success | `#16a34a` | 성공, 완료 상태 |
| Danger | `#dc2626` | 오류, 삭제 |
| Warning | `#f59e0b` | 경고, 대기 상태 |
| Gray 50 | `#f9fafb` | 배경 |
| Gray 100 | `#f3f4f6` | 카드 배경 |
| Gray 200 | `#e5e7eb` | 테두리 |
| Gray 300 | `#d1d5db` | 입력 테두리 |
| Gray 500 | `#6b7280` | 보조 텍스트 |
| Gray 700 | `#374151` | 본문 텍스트 |
| Gray 900 | `#111827` | 제목 텍스트 |

### 타이포그래피

| 요소 | 크기 | 굵기 | 용도 |
|------|------|------|------|
| H1 | 24px | 600 | 페이지 제목 |
| H2 | 20px | 600 | 섹션 제목 |
| H3 | 16px | 600 | 카드 제목 |
| Body | 14px | 400 | 본문 텍스트 |
| Small | 13px | 400 | 보조 텍스트 |
| XSmall | 12px | 400 | 라벨, 캡션 |

### 간격

| 이름 | 값 | 용도 |
|------|-----|------|
| xs | 4px | 요소 내 간격 |
| sm | 8px | 작은 간격 |
| md | 12px | 기본 간격 |
| lg | 16px | 섹션 간격 |
| xl | 20px | 큰 섹션 간격 |
| 2xl | 24px | 페이지 여백 |

```
┌─────────────────────────────────────────────────────────┐
│  Header (로고, 사용자 정보, 로그아웃)                     │
├─────────┬───────────────────────────────────────────────┤
│         │                                               │
│ Sidebar │              Main Content                     │
│         │                                               │
│ 메뉴    │    ┌─────────────────────────────────────┐   │
│         │    │ Page Header (제목, 액션버튼)          │   │
│ - 대메뉴 │    ├─────────────────────────────────────┤   │
│   - 하위 │    │                                     │   │

│         │    │                                     │   │
```

### 사이드바 메뉴

#### 군수담당자 (admin)

```
📋 대시보드
👥 사용자 관리
   - 사용자 목록
   - 사용자 등록
   - 일괄 등록
👔 계급 관리
   - 진급 처리
📦 품목 관리
   - 카테고리 관리
   - 품목 목록
💰 포인트 관리
   - 포인트 지급
   - 지급 이력
🏪 판매소 관리
🪡 체척업체 관리
```

#### 판매소 담당자 (sales_office)

```
📋 대시보드
📦 주문 관리
   - 주문 목록
   - 배송 처리
🛒 오프라인 판매
↩️ 반품 처리
📊 재고 관리
📍 배송지 관리
📈 통계
```

#### 체척업체 담당자 (tailor_company)

```
📋 대시보드
🎫 체척권 관리
   - 체척권 현황
   - 체척권 등록
```

#### 일반사용자 (general)

```
🛍️ 쇼핑
   - 완제품
   - 맞춤피복
📋 내 주문
💰 내 포인트
👤 내 정보
```

## 반응형 디자인

### 브레이크포인트

| 이름 | 범위 | 사이드바 |
|------|------|---------|
| Desktop | 1024px+ | 표시 |
| Tablet | 768-1023px | 축소/숨김 |
| Mobile | < 768px | 햄버거 메뉴 |

### 모바일 대응

- 테이블 → 카드 리스트 변환
- 사이드바 → 드로어 메뉴
- 터치 친화적 버튼 크기 (최소 44px)
